## Note on model-spectrum normalization

The spectra generated by `make_physical_model_csvs.py` include a fixed factor of `1e45` in the rate calculation for numerical convenience. This factor is not a physical unit conversion; it is only used to keep the output values in a convenient numerical range. Therefore, the exported `dRdE` values should be interpreted as arbitrarily normalized recoil-spectrum templates rather than absolute physical recoil rates:

```python
rate = 1e45 * prefactor * F2 * eta

## Model assumptions 

- The signal model implemented in "make_physical_model_csvs.py" is a standard spin-independent (SI) elastic WIMP–xenon scattering model. The generated spectra are intended to serve as input templates for the XENON1T approximate nuclear-recoil likelihood.

## Assumptions used in the current model

The current implementation uses the following assumptions:

- Target nucleus: xenon with mass number `A = 131`
- Nuclear mass: `M_NUCLEUS_GEV = A_XE * 0.9315`
- Proton mass: `M_PROTON_GEV = 0.938272 GeV`
- Interaction model: standard spin-independent elastic scattering
- Reference cross section: `SIGMA_SI_REF = 1e-45 cm^2`
- Halo model: Standard Halo Model with
  - `RHO_CHI = 0.3 GeV/cm^3`
  - `V0 = 220 km/s`
  - `VE = 232 km/s`
  - `VESC = 544 km/s`
- Nuclear form factor: Helm form factor
- Energy range of exported spectra: `1` to `70 keV` in `1 keV` steps
- Mass grid currently exported: `5, 10, 20, 30, 40, 50, 70, 100, 150, 200 GeV`
